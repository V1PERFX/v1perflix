## Como usar

No terminal:

```bash
# Faça um clone deste repositório
$ git clone https://github.com/V1PERFX/v1perflix.git

# Entre no repositório
$ cd v1perflix

# Instale as dependências
$ npm install

# Execute
$ npm run dev
```
